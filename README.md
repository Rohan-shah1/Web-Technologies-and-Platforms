## Description
This project is a comprehensive collection of HTML laboratory exercises completed as part of the college curriculum. It demonstrates the mastery of various HTML modules including:
- **Module 1-2:** Fundamentals, Text, Links & Lists (`index.html`, `about.html`)
- **Module 3:** Images & Media (`media.html`)
- **Module 4:** Page Structure & Semantic HTML (`index.html`)
- **Module 5:** HTML Forms (`forms.html`)
- **Module 6:** HTML Tables (`tables.html`)
- **Module 8:** Advanced HTML Features like Details/Summary (`faq.html`)

## How to Run
1. Clone this repository or download the files.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
3. Navigate through the site using the menu links to view different lab exercises.

## Project Structure
- `index.html`: Main landing page with semantic structure.
- `about.html`: User profile and skills.
- `media.html`: Audio, video, and image examples.
- `forms.html`: A complete admission form.
- `tables.html`: A complex student result table.
- `faq.html`: FAQ section using interactive HTML tags.
