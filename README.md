# FreshCart - Organic & Fresh Grocery Store

A pixel-perfect, responsive frontend implementation of the FreshCart e-commerce template. This project demonstrates a modern, clean user interface for an online grocery store.

## Features
- **Responsive Design**: Fully adaptive layout for mobile, tablet, and desktop.
- **Modern UI/UX**: clean typography, hover effects, and smooth transitions.
- **Dynamic Components**:
  - **Sticky Navbar**: With mobile toggle and active states.
  - **Hero Section**: Full-width banner with promotional copy.
  - **Product Cards**: Detailed cards with sale badges, ratings, and action buttons.
  - **Mega Footer**: Comprehensive 4-column footer layout.
- **Asset Integration**: High-quality images for categories, products, and banners.

## Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, CSS Grid.
- **JavaScript**: Vanilla JS for DOM manipulation (mobile menu, scroll effects).
- **FontAwesome**: For icons.
- **Google Fonts**: 'Outfit' typeface.

## How to Run
1. Clone the repository:
   ```bash
   git clone -b Freshcart https://github.com/Rohan-shah1/Web-Technologies-and-Platforms.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Web-Technologies-and-Platforms/FreshCart
   ```
3. Open `index.html` in your browser.
