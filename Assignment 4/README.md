# Student Management System Interface
### Advanced HTML & CSS Assignment

## Project Description
A responsive front-end interface for a Student Management System. This project demonstrates advanced HTML5 and CSS3 techniques to create a modern, interactive web application layout without JavaScript.

## Features & Technologies Used
- **HTML5**: Semantic elements, Forms, Tables.
- **CSS3 Advanced**:
    - **Flexbox**: Used for Navigation bars, Center alignment, and Layouts.
    - **CSS Grid**: Used for Dashboard Layout (Sidebar + Content) and Statistics Cards.
    - **Variables**: CSS Custom Properties for consistent theming.
    - **Positioning**: `sticky` Navbar and Sidebar.
    - **Animations**: Keyframe animations for page load (`fadeIn`, `slideUp`) and hover transitions.
    - **Responsiveness**: Media queries for Tablet (1024px) and Mobile (768px) views.

## Pages Included
1. **Landing Page (`index.html`)**: Hero section with Call-to-Action.
2. **Login Page (`login.html`)**: Centered login card with focus effects.
3. **Dashboard (`dashboard.html`)**: Admin view with stats and sticky sidebar.
4. **Student List (`manage.html`)**: Data table view with responsive layout.

## Setup & Usage
1. Open `index.html` in any modern web browser.
2. Navigate effectively using the top menu or dashboard sidebar.
3. Resize the browser window to see responsive adaptations.
