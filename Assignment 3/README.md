# Student Management System - Assignment 3

## Description
This static web interface represents a Student Management System designed using HTML5 and CSS3. The system layout relies on normal document flow, block/inline-block elements, and standard box model properties, strictly avoiding Flexbox, Grid, and Positioning as per assignment constraints.

## Pages
1. **Home Page (`index.html`)**: Introduction to the system.
2. **Login Page (`login.html`)**: User authentication form.
3. **Dashboard Page (`dashboard.html`)**: Overview of student statistics.
4. **Student List Page (`list.html`)**: Tabular view of student records.
